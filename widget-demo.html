Omni AI Widget Demo

Quick Start

Add this single line of code to your website, just before the closing </body> tag:

<script src="https://your-domain.com/widget.js"></script>

Try it now! Look at the bottom-right corner of this page to see the widget in action.

Customization Options

Customize the widget using data attributes:

<script src="https://your-domain.com/widget.js" data-api="https://your-api.com" data-position="right" data-color="#6366f1" data-title="Support" data-greeting="Hi! How can I help?" ></script>
Attribute Default Description
data-api Omni API Your backend API URL
data-position right Widget position: left or right
data-color #6366f1 Primary color (hex)
data-title Omni AI Header title text
data-greeting Hi! How can I help you today? Initial greeting message

Features

The embeddable widget includes:

  • Lightweight (~5KB gzipped)
  • No dependencies
  • Mobile responsive
  • Persistent session across page reloads
  • Smooth animations
  • Customizable colors and text
  • Accessible (keyboard navigation, ARIA labels)

Self-Hosting

To self-host the widget:

  1. Download widget.js from your Omni AI dashboard
  2. Upload to your CDN or static file server
  3. Update the script src to your hosted URL
  4. Configure data-api to point to your backend
